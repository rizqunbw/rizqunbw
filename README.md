# Muhammad Rizqu Nabawi Portfolio

Portfolio website built with `React` and `Vite` for `rizqunbw.vercel.app`.

## Focus

This site presents Muhammad Rizqu Nabawi as:

- `Game Owner`
- `IT Operations & Server Administrator`
- `IT Support / Technical Support` candidate

## Routes

- `/` main portfolio
- `/moneytracker` project detail page
- `/rfitemcrop` project detail page

## Development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Deploy

Deploy to Vercel using the root project directory. The site is configured as a static SPA with rewrites in `vercel.json`.
